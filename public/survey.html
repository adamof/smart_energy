Background

As part of my honours project I have developed a software which provides the same functionality as the meters that will be going into homes and it will be used as part of a major project - IDEAL project at the University of Edinburgh. This project will focus on examining the effect of providing personalised behavioural feedback to occupants: explicitly identifying the behaviours (e.g., showering and bathing, room heating, major appliance use) that account for energy use/expenditure; and how these could be reduced.

My project

As mentioned above, this web application is supposed to mimic the functionality provided by the smart meters that will be installed as part of the Smart Meter programme. That functionality includes displaying information about the energy(gas and electricity) usage such as:

  • energy usage
  • energy cost
  • carbon footprint
